# 100 Days of Python Bootcamp

A growing collection of Python projects and exercises completed as part of a 100-day Python coding bootcamp.
The curriculum progresses from fundamentals to advanced topics, with hands-on projects in:

- Automation
- Game development
- Web development
- Data science
- Machine learning

The bootcamp also introduces popular Python libraries and frameworks, including:
**Selenium, Beautiful Soup, Requests, Flask, Pandas, NumPy, Scikit-learn, Plotly, and Matplotlib.**

## Days Completed

- **Day 1**: Band name generator
- **Day 2**: BMI calculator, tip calculator
- **Day 3**: Pizza delivery, Treasure Island game
- **Day 4**: Heads or tails, Rock-Paper-Scissors
- **Day 5**: FizzBuzz, password generator, loops
- **Day 6**: Hurdle loop challenge
- **Day 7**: Hangman game
- **Day 8**: Caesar cipher, functions, love calculator
- **Day 9**: Blind auction, dictionaries, grades
- **Day 10**: Leap year, calculator, docstrings
- **Day 11**: Blackjack game

## How to Run

Run any Python file with:

```bash
python day-XX/filename.py
```

Requires Python 3.x.
